Управление интуитивнопонятное:

Жирным шрифтом обозначены узлы, которые содержат подстроки.

space - открывает/закрывает подстроки.

del - удаляет данный узел.

enter - и двойной клик открывает окно редактирования.

esc - закрывает окно редактирования.

enter при открытом окне редактирования сохраняет результат и закрывает окно редактирования.

Кнопка "Сохранить" - сохраняет данному узлу значение, которое находится в окне редактирования.

Кнопка "Удалить" - удаляет данный узел и значение окна редактирования никак не влияет.

Кнопка "Добавить подстроку" - добавит подстроку со значнием окна редактирования.

Есть есть один момент. Допустим Вы хотите сделать несколько подстрок у данного элемента.

Вы можете использовать длительное нажатие enter, чтобы открыть окно редактирования повторно. Либо короткое нажатие, чтобы не открывать окно.

Попробуйте и почувствуете.

Используются следующие классы:

  1. tree-space - контейнер для дерева
  2. tree-space__line - пункт в дереве
  3. tree-space__list - список пунктов
  4. tree-space__parent - пункт, содержащий подпункты
  5. tree-space__change - блок редактирования
  6. change__field - поле блока редактирования
  7. change__button - копки блока редактирования. Не используются в стилях, но создан этот класс именно для удобного узменения стилей кнопок
  8. change__save - класс кнопки сохранения
  9. change__add - класс кнопки добавления подстроки
  10. change__del - класс кнопки удаления

Используются следующие id:

  1. tree-space__change_№ - вместо "№" - цифра. Идентификатор блока изменений
  2. change__field_№ - идентификаторы соответствующих элементов блока изменений
  3. change__save_№ - идентификаторы соответствующих элементов блока изменений
  4. change__add_№ - идентификаторы соответствующих элементов блока изменений
  5. change__del_№ - идентификаторы соответствующих элементов блока изменений

Для сохранения в localStorage:

  • tree_№ - вместо "№" - цифра.
